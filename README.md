# Covid-19 Dashboard

### Requirements
- #### Laravel 7.x
- #### PHP 7.x+
- #### composer

## How to Use :
### Add Page
- #### Click on the Add page button to start adding countries
### Edit Page
- #### Click on the pen icon to start editing country details
### View Covid details
- #### Click on the country name ie. "Saudi Arabia"
### Sorting Table
- #### Click on the column head you desire to sort by
